printf command in bash


$ type printf
printf is a shell builtin
$


"printf prints a formatted string to the standard output.
Its roots are in the C programming language, which uses
a function by the same name.  It is a handy way to
produce precisely-formatted output from numerical or
textual arguments."


Syntax
======
printf FORMAT [ARGUMENT]...
printf OPTION


Options
-------
FORMAT	    FORMAT controls the output, and defines the way that
            the ARGUMENTs will be expressed in the output.
            See the Format section, below.

ARGUMENT    Each ARGUMENT will be inserted into the formatted
            output according to the definition of FORMAT.

--help	    Display a help message, and exit.

--version   Display version information, and exit."
--www.computerhope.com/unix/uprintf.htm





Example
=======
$ printf "foo %s\n" "bar"
foo bar
